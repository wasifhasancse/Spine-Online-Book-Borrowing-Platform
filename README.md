# Spine

Spine is an online book borrowing platform where users can browse books, filter by category, check availability, view detailed information, and manage their account from a responsive web interface.

## Live URL

Visit the live site: [Spine - Online Book Borrowing Platform](https://spine-online-book-borrowing-platfor.vercel.app)

```text
https://spine-online-book-borrowing-platfor.vercel.app
```

## Purpose

The project was built to create a simple and practical online library experience. It focuses on helping readers explore books quickly, find what they need with filters and search, and handle account-related actions like sign in, sign up, and profile updates.

## Technologies Used

- Next.js
- React
- Tailwind CSS
- Better Auth
- MongoDB
- HeroUI

## 3 Key Features

- Category-based browsing and title search so users can quickly find books they want.
- Book details view with stock, rating, pricing, and borrowing action.
- Authentication system with email/password login, Google sign-in, and profile management and update.
